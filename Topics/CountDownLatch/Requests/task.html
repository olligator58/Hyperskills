You are in charge of a system that handles requests from clients. Each request is handled by a separate thread. The main thread of the system should wait until all requests are completed before it continues execution.

Your task is to simulate this scenario using the CountDownLatch class in Java. For simplicity, assume that each request multiplies a value by 2. The input is the count of threads. The output is the final value of the data.


Sample Input:
3

Sample Output:
8



Memory limit: 256 MB
Time limit: 5 seconds
