Implement a thread-safe cache that utilizes a ReentrantReadWriteLock to manage access. The cache should support the following operations:

  • add(): Increment the cache value by 1.
  • get(): Return the current value of the cache.

The cache should employ a ReentrantReadWriteLock to ensure thread safety for these operations. The add operation should acquire the write lock before modifying the cache, while the get operation should acquire the read lock before accessing the cache.

The input to the program is the number of times the add operation is performed. The output should be the final cache value., which should be equal to the count of add operations performed.


Sample Input:
8

Sample Output:
24



Memory limit: 256 MB
Time limit: 5 seconds
