Your assignment involves creating a "Worker" class within a multithreaded context. The program will generate a predetermined number of threads. Upon execution, each thread will increment a shared counter by a value "n" and then terminate its operation. The shared counter's initial value is set to 0. The final value of this shared counter should be provided as the result.

The program's input is the total number of threads that need to be created. The expected output is the correct final value of the shared counter.


Sample Input:
3

Sample Output:
6



Memory limit: 256 MB
Time limit: 5 seconds
