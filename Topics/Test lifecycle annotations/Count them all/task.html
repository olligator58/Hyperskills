Given the following test class:

class BetaTest {

    @BeforeAll
    static void method3() {
        System.out.println("Hello");
    }

    @BeforeEach
    void method2() {
        System.out.println("Hello");
    }

    @AfterAll
    static void method4() {
        System.out.println("Hello");
    }

    @AfterEach
    void method5() {
        System.out.println("Hello");
    }

    @Test
    void method6() {
        System.out.println("Hello");
    }

    @Test
    void method7() {
        System.out.println("Hello");
    }
}

How many times the word "Hello" will be printed?
